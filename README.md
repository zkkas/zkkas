## Featured Projects

### 🖥️ Online Judge — OBI Practice Platform
A full-stack competitive programming judge modeled after LeetCode/Codeforces, focused on Brazilian Informatics Olympiad problems.

- Docker-based sandboxed code execution with real-time verdict delivery via **Socket.io**
- JWT authentication, seeded problem set with verified test cases
- Built with **Next.js, Node.js/Express, PostgreSQL, Prisma**
- Web scraper targeting the official OBI practice site for problem ingestion

---

### 🛒 PC Parts Rating & Review Platform
Full-stack platform for rating and reviewing PC components, built as a group database course project.

- REST API with **Node.js, Express, Prisma ORM**
- **PostgreSQL** database with relational schema design
- Deployed with Neon + Render + Vercel

---

### 📡 SADA-BR — Real-Time Anti-Aircraft Radar Simulation
Real-time radar simulation using live aircraft data from the OpenSky Network API.

- Intercept physics engine (haversine distance, quadratic intercept equations)
- Leaflet map with Canvas overlay, threat classification, missile launch simulation
- OAuth2 authentication, military HUD aesthetic
- Built with **Python/Flask**

---

### 🏢 NEXA — Multi-Tenant B2B SaaS API
REST API architecture for a B2B SaaS platform targeting micro businesses.

- Multi-tenant isolation middleware, role-based permissions
- Three-tier subscription plan structure
- Built with **Node.js, TypeScript, Express, Prisma, PostgreSQL, Redis**
